🌊 My Global Dive Sites

Scuba diving has been one of the most incredible ways for me to connect with the underwater world and witness biodiversity up close. Below is a map of the dive sites I’ve explored around the world — from the reefs of Koh Tao to the shores of Cabo. These are my personal ratings based on what I observed during my dives — particularly the biodiversity (variety and abundance of marine life) and coral vibrance (color, density, and health of coral ecosystems).

I’m always excited to explore more dive sites in the future and hope to expand this map as I keep diving! 🐠💦

library(leaflet)

# Create a data frame of your dive sites
dives <- data.frame(
  location = c("Great Barrier Reef, Australia", "Andaman & Nicobar Islands, India", "Fiji",
               "Bahamas", "Cancun, Mexico", "Hawaii (Big Island)", "Hawaii (Kauai)",
               "Sri Lanka", "Cabo, Mexico", "Jamaica", "Goa, India", "San Diego, USA",
               "Koh Tao, Thailand", "Menjangan, Indonesia"),
  lat = c(-16.9186, 11.7401, -17.7134, 25.0343, 21.1619, 19.5429, 22.0964,
          7.8731, 22.8894, 18.1096, 15.2993, 32.7157, 10.0982, -8.1086),
  lng = c(145.7781, 92.6586, 178.0650, -77.3963, -86.8515, -155.6659, -159.5261,
          80.7718, -109.9159, -77.2975, 74.1240, -117.1611, 99.8382, 114.5420),
  biodiversity = c(5, 3, 3, 5, 5, 5, 5, 3, 5, 3, 2, 2, 5, 5),
  coral = c(2, 4, 5, 3, 4, 2, 5, 4, 3, 3, 2, NA, 5, 5),
  stringsAsFactors = FALSE
)

# Create popup content
dives$popup <- paste0(
  "<strong>", dives$location, "</strong><br/>",
  "🧬 Biodiversity Rating: ", dives$biodiversity, "/5<br/>",
  "🌈 Coral Vibrance: ", dives$coral, "/5"
)

# Generate the interactive map
leaflet(data = dives) %>%
  addTiles() %>%
  addCircleMarkers(
    ~lng, ~lat,
    popup = ~popup,
    radius = 6,
    color = "deepskyblue",
    fillOpacity = 0.8
  )
